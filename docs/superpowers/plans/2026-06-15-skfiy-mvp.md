# skfiy MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox syntax for tracking.

**Goal:** Build a macOS desktop Computer Use prototype that can observe the desktop, activate Ghostty, type safe commands, capture screenshots, and show progress in a floating desktop pet window.

**Architecture:** Electron provides the floating always-on-top UI and task shell. A TypeScript core owns app state, risk policy, and task orchestration. A Swift command-line helper performs macOS operations through ScreenCaptureKit/screencapture, Accessibility, and CoreGraphics.

**Tech Stack:** Electron, Vite, React, TypeScript, Vitest, Swift, macOS Accessibility/Screen Recording APIs.
